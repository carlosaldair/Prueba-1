¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML, utilizado para estructurar y presentar contenido en la web.

Define la estructura de una página web usando etiquetas para organizar texto, imágenes, videos, formularios, etc.

Publicado el 25/07/2025 por UMG-Guastatoya

Introducción a CSS3

CSS3 es un lenguaje de diseño utilizado para definir la apariencia visual de páginas web escritas en HTML.

Puedes cambiar colores, tamaños de texto, fondos, bordes, y mucho más.
